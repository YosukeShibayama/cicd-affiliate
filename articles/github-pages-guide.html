GitHub Pages

GitHub Pagesで静的サイトを公開する基本

更新予定: 記事本文は別ブランチで作り込み予定

GitHub Pagesは、HTML/CSS/JavaScriptだけの静的サイトを公開する練習に向いています。 このサイトでは、Pull RequestとGitHub Actionsも合わせて学びます。

学習する流れ

  1. featureブランチで変更する
  2. Pull Requestを作る
  3. CIでlintを確認する
  4. レビュー後にmainへマージする
  5. deploy workflowでGitHub Pagesへ公開する
